> **Note:** It may take a while for the Codespace to spin up. It will be ready when the terminal asks for your Vonage API key.

The steps in the terminal (look below) will create a Vonage application and set up the webhook endpoints so you don't have to do it manually.

You can find your `API KEY` and `API SECRET` in the <a href="https://dashboard.vonage.com" target="_blank">Vonage Dashboard</a>.
