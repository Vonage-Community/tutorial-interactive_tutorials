# Create a Vonage Account

To use the Vonage Video API, you need a Vonage account. In this step, you will create one if you don't already have it.

{% steps %}

1. Navigate to the [Vonage API Dashboard](https://dashboard.nexmo.com/).

2. Click **Sign Up** and complete the registration form.

3. Verify your email address and log in to the dashboard.

{% /steps %}

{% aside type="tip" %}
If you already have a Vonage account, simply log in to the [dashboard](https://dashboard.nexmo.com/) and proceed to the next step.
{% /aside %}

You now have access to the Vonage API Dashboard where you can manage your applications, keys, and usage. Next, you'll create an application with Video capabilities enabled.
