# Prerequisites

Before you begin, make sure you have the following tools installed on your machine.

## Required

- **Node.js** version 22
- **yarn** package manager

## Optional (Recommended)

- **nvm** — Node Version Manager, useful for switching between Node.js versions

{% aside type="tip" %}
If you use nvm, you can install and switch to Node.js 22 with:

```sh
nvm install 22
nvm use 22
```
{% /aside %}

You can verify your Node.js version by running:

```sh
node --version
```

You should see output like `v22.x.x`.

To check that yarn is available:

```sh
yarn --version
```

{% aside type="caution" %}
The app requires Node.js version 22. Other versions may not work correctly.
{% /aside %}

With the prerequisites in place, you're ready to set up your Vonage account.
