title: Run the Application
description: Time to test the application
---

Now is the time to test the application and create a multiparty video call, start, stop and view an archive.

There is a `PREVIEW READY` link in the terminal below with a URL. Open that link in 3 or more windows.

> Note: To prevent audio feedback, please use headphones or mute your computer.

Click the `Start Archive` button and verify that the start archiving indicator shows on your video. After a few seconds, press the `Stop Archive`. A link to view to video recording will appear. Click that link and if it is ready, you will be able to see the recording.
