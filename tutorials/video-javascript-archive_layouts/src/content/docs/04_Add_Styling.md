---
title: Add Styling
description: Let's add some style to the application
---

Add this code snippet to the `app.css` file in the `css` folder that will put a white background for the layout controls (`/* ⌄⌄⌄ Add CSS for the layout-controls section ⌄⌄⌄ */`):

```css
#layout-controls {
  background-color: white;
}
```
