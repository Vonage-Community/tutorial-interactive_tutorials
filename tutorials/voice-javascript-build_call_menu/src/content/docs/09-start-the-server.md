# Start the Server

Now that all the code is in place, start the Express server from the terminal. Make sure you are inside the `project/` directory first:

```sh
cd project && npm start
```

The `--watch` flag means the server will automatically restart whenever you save `index.js`.

Open the **Preview** browser tab and visit:

```
http://localhost:3000
```

If the server is running correctly you will see:

```
I'm listening!
```

Leave the server running for the next steps.
