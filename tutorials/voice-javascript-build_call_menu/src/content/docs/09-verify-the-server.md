# Verify the Server

The Codespace starts the Express server automatically. In the **Ports** tab, find port **3000** and open its **Forwarded Address**.

If the server is running correctly, the page displays:

```
I'm listening!
```

The server uses `node --watch`, so it automatically reloads the completed `index.js` whenever you save the file. Leave it running for the next steps.
