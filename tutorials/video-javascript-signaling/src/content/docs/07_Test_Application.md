---
title: Run the Application
description: Time to test the application
---

Now is the time to test the application and use the text chat.

Refresh the window/tab you opened the application in earlier to make sure all the code changes have been applied.

Open that application link in another window/tab so that you have two participants.

> Note: To prevent audio feedback, please use headphones or mute your computer.

Send a chat message in one window/tab and then navigate to the other window/tab and send another chat message. Verify that the messages appear in both windows/tabs.
