---
title: Setup Application
description: Create an application.
---

In the terminal below, you should see a set up script running asking for your `API key`. Complete the steps in the set up script. This will create an Application with Video functionality and create the Webhook endpoints needed.

> Note: You can get your credentials in the <a href="https://dashboard.vonage.com/settings" target="_blank">Vonage dashboard Settings Page</a>

If successful, you should see `APPLICATION READY` with a URL.
