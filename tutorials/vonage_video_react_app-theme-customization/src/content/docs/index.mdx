The Vonage Video React App ships with a fully themeable UI. Every color, font size, font weight, and border radius can be changed without touching any source code — all through a single file at the project root: **`designTokens.json`**.

In this tutorial you will:

- Understand the structure of `designTokens.json` and how it drives the app's visual theme
- Customize color palettes for both light and dark modes
- Adjust typography settings such as font family, sizes, and weights
- Sync your changes and verify them in the running app

> **Prerequisites:** This tutorial assumes the app is already running locally. If you haven't set it up yet, complete the [Vonage Video React App — Local Setup](../vonage_video_react_app-local-setup/) tutorial first.
