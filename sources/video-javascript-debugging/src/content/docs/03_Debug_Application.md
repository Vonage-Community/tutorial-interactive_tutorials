---
title: Debug Application
description: Debug an application.
---

Click the `APPLICATION READY` link to open in multiple windows or tabs.

The demo is a video application that you can start and stop an archive. After using the application for a little while, copy the GrapQL query and click the `Open GraphiQL IDE` button.

Paste the query into the editor and click the play button. This will use the Vonage Video Insights API to obtain information about your Vonage Video applications and sessions.

You can explore the GraphiQL IDE and test out different queries.
