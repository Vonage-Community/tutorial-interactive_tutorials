In this tutorial, you will configure and run the **vonage-video-react-app** — a full-stack reference application built with React and Node.js that demonstrates real-time video communication using the Vonage Video API.

The app includes features such as:

- One-to-one and multiparty video calls
- Screen sharing
- Chat messaging
- Recording and archiving

By the end of this tutorial, you will have the application running locally on your machine, ready for development and exploration.

{% aside type="tip" %}
The vonage-video-react-app is an open-source reference implementation. You can find the source code at [github.com/Vonage/vonage-video-react-app](https://github.com/Vonage/vonage-video-react-app).
{% /aside %}
