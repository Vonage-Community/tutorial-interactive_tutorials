---
title: Verify server
description: Steps to get the server running.
---

To make sure the server is running correctly, you will need to verify that it returns a set of credentials that allows a user to join a video call.

In the terminal below, you will see `APPLICATION READY` with a URL.

Open the URL in a new tab.

At the end of the URL, add `/session`.

> Note: If there is already a `/` at the end of the URL, just add `session`

The `applicationId`,  `sessionId` and `token` should appear on the screen.
