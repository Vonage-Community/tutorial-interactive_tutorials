title: Flue Framework
description: Explaining some basic concepts.
---

<a href="https://flueframework.com" target="_blank">Flue</a> gives us a framework, written in TypeScript, where we can place an LLM into an environment that provides a set of capabilites to do various tasks and achieve goals.

The environment is called a <a href="https://flueframework.com/docs/concepts/agents/#what-is-a-harness" target="_blank">Harness</a> and the LLM inside that harness is an <a href="https://flueframework.com/docs/concepts/agents" target="_blank">Agent</a>.

The application we are building allows our Gemini powered Agent to pretend to be Leonardo da Vinci and the Harness allows us to make phone calls to communicate.
