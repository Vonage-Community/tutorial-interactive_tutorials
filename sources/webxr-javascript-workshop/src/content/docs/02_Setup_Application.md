title: Setup Application
description: Create an application.
---

> Note: Please make sure to redeem the coupon code to your Vonage account's <a href="https://dashboard.vonage.com/coupons" target="_blank">dashboard</a> before continuing. It is needed to purchase a phone number.

In the terminal below, you should see a set up script running asking for your `API key`. Complete the steps in the set up script. This will create an Application with Voice functionality, purchase a phone number, and create the needed Webhook endpoints.

> Note: You can get your credentials in the <a href="https://dashboard.vonage.com/settings" target="_blank">Vonage dashboard Settings Page</a>

If successful, you should see `APPLICATION READY` with a URL. Click the link to open in a new window or tab.
