title: Run the Application
description: Time to test the application
---

Now is the time to test the application.

Refresh the window/tab you opened the application in earlier to make sure all the code changes have been applied.

Open that application link in a few more windows/tabs so that you have mulitple participants.

> Note: To prevent audio feedback, please use headphones or mute your computer.

Click the `Start Archive` button and verify that the start archiving indicator shows on your video. Then select the different layout options in the dropdown waiting a few seconds between each selection. When ready, press the `Stop Archive`. A link to view to video recording will appear. Click that link and if it is ready, you will be able to see the recording with all the layout changes you made.
