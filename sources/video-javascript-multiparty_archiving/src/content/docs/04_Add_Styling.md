---
title: Add Styling
description: Let's add some style to the application
---

Add this code snippet to the `app.css` file in the `css` folder that will put a white background with 5px padding to the link to view the archive (`/* ⌄⌄⌄ Add CSS for the archiveLink element ⌄⌄⌄ */`):

```css
#archiveLink > a {
  padding: 5px;
  background-color: white;
}
```
